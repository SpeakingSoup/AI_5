# Лабораторная работа №5: "Рекуррентные и одномерные сверточные нейронные сети"

Данный репозиторий содержит файлы выполненной лабораторной работы №5

## Содержание

- *lite* – задание уровня Lite. Цель - провести ряд экспериментов с разными параметрами.
- *lite_pytorch* – задание уровня Lite, выполненное с помощью библиотеки pytorch. 
- *pro* – задание уровня Pro. Цель - создать модель, определяющую болезни.
- *ultra_pro* – задание уровна Ultra Pro. Цель - создать модель, определяющую авторов.
